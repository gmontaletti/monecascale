# Format a \`moneca_bipartite\` object

One-line summary of a bipartite fit suitable for \`cat()\` contexts.

## Usage

``` r
# S3 method for class 'moneca_bipartite'
format(x, ...)
```

## Arguments

- x:

  An object of class \`"moneca_bipartite"\`.

- ...:

  Further arguments, currently ignored.

## Value

A single character string describing the bipartite fit.
